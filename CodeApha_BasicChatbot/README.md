# CodeAlpha_BasicChatbot
🤖 Basic Chatbot (CodeAlpha Internship Task 4)

This is a simple text-based chatbot developed in Python as part of the CodeAlpha Internship.

The chatbot interacts with the user by responding to basic inputs like greetings and simple questions using predefined replies.

---

# Features

- Responds to inputs like:
  - "hello"
  - "hi"
  - "how are you"
  - "bye"
- Predefined rule-based responses
- Continuous interaction using loop
- Exits when user types "bye"
- Simple console-based chatbot
- Beginner-friendly implementation

---

# Concepts Used

- Python Basics
- Functions
- Loops (while loop)
- Conditional Statements (if-elif-else)
- Input/Output

---

# How to Run

1. Install Python
2. Download or clone this repository
3. Run the file:

python chatbot.py
