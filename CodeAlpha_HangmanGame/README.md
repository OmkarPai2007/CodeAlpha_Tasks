# CodeAlpha_HangmanGame
🎮 Hangman Game (CodeAlpha Internship Task 1)

This is a simple text-based Hangman Game developed in Python as part of the CodeAlpha Internship.

The player has to guess a randomly selected word one letter at a time. The player is allowed a maximum of 6 incorrect guesses.

---

# Features

- Random word selection
- 5 predefined words
- Maximum 6 incorrect attempts
- Simple console-based gameplay
- Beginner-friendly implementation

---

# Concepts Used

- Python Basics
- Lists & Strings
- Loops (while loop)
- Conditional Statements (if-else)
- Random Module

---

# How to Run

1. Install Python
2. Download or clone this repository
3. Run the file:

python hangman.py
