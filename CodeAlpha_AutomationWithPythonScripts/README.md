# CodeAlpha_EmailExtractor

📧 Email Extractor (CodeAlpha Internship Task 3)

This is a simple Python-based Email Extractor developed as part of the CodeAlpha Internship.

The program reads a text file, extracts all valid email addresses using regular expressions, removes duplicate entries, and saves them into a new file.

---

# Features
- Extracts emails from a text file
- Removes duplicate email addresses
- Supports multiple email formats
- Simple and console-based
- Beginner-friendly implementation

---

# Concepts Used
- Python Basics
- File Handling
- Regular Expressions (re module)
- Sets (for duplicate removal)
- Functions

---

# How to Run
1. Install Python
2. Download or clone this repository
3. Run the file:
   python main.py
4. Enter input and output file names when prompted
